# Mini-Game Lab — 抖音小游戏开发实验室

> 一个 LLM 研究者的副业尝试：在 6 个月内用 Cocos Creator + 抖音小游戏平台，独立开发并运营 2-3 款休闲小游戏。

---

## 项目愿景

- **直接目标**：通过多款矩阵化小游戏获得可持续的副业收入（目标：6 个月内月入 5,000-30,000 元）
- **衍生价值**：将开发过程沉淀为可发表的技术博客、经验复盘、研究素材
- **方法论**：单兵开发 + 数据驱动 + 多产品矩阵 + 内容反哺

---

## 技术栈

| 维度 | 选型 | 理由 |
|------|------|------|
| 引擎 | Cocos Creator 2.4.13 LTS | TS 零迁移 / 抖音适配最好 / 模板资源最多 |
| 语言 | TypeScript | 类型安全 / 工程化 |
| 渲染 | Canvas 2D | 休闲品类够用，规避 WebGL 适配 |
| 后端 | 抖音云开发 / 微信小游戏同款 | 学生单人 / 无自建服务器 |
| 数据 | 抖音云数据库 / 本地存储 | 冷启动期本地即可 |
| 广告 | 穿山甲 Pangle SDK | 抖音官方广告 SDK |
| 包管理 | npm + Cocos 内置 | 不引入额外复杂度 |

## 关键指标追踪

每款游戏追踪 6 个核心指标：

| 指标 | 目标 | 数据来源 |
|------|------|----------|
| DAU（日活） | M3 ≥ 100 / M6 ≥ 1000 | 抖音开放平台后台 |
| 次留 | ≥ 30% | 抖音数据看板 |
| 7 留 | ≥ 12% | 抖音数据看板 |
| 广告 ARPU | ≥ 0.05 元 | 穿山甲后台 |
| 广告 eCPM | ≥ 30 元 | 穿山甲后台 |
| 视频挂载转化 | ≥ 2% | 抖音视频数据 |

每周末记录到 `weekly-reports/YYYY-week-XX.md` 中。
